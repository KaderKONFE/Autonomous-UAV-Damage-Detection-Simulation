# Autonomous UAV Damage Detection Simulation

This repository contains the software architecture for an autonomous UAV simulation designed to detect structural cracks on building columns after an earthquake.

## Tech Stack
* **OS:** Linux Ubuntu
* **Middleware:** ROS 2
* **Flight Controller:** PX4 Autopilot
* **Simulator:** Gazebo

## My Contributions
* Developed the ROS 2 software nodes and flight control architecture.
* Integrated the PX4 Autopilot for autonomous flight routes.
* Created and configured the 3D simulation environment in Gazebo.
